My Projects

StelStay Properties

StelStay Properties Information System is a student accommodation management application developed by SBSJ Technologies. This project was part of my curriculum last year, where we utilized React Admin and Vite, with Pocketbase for the backend. The system focuses on simplifying the lease application process and efficient rent invoicing. Key features include a lease application process where students can apply for accommodation and administrators manage applications for approval and room assignment, and a rent invoicing feature that generates and manages monthly rent invoices, ensuring a smooth payment process. Group members included Joshua Sayers, Bafana Madume, Scott Stewart, and myself.
